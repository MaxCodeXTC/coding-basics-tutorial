Semicolons

Semicolons are required if we want to separate statements in single line i.e.


alert('Stay'); alert('Home');
// but this will also work:
alert('Stay')
alert('Home')
// but this will NOT WORK:
alert('Stay') alert('Home')

AS you can see they are not required. However, some people believe that it is a good practice to put them in at the end of each statement.

Variables - declaration / assignment / usage

We can easily grasp the concept of a “variable” if we imagine it as a “box” for data, with a uniquely-named sticker on it. For instance, the variable message can be imagined as a box labeled "message" with the value "Hello!" in it: We can put any value in the box.

In order to use variables, we have to:

  1. Declare it:
    2. 
var message;
  2. Assign value to it: 
    3. 
    message = 'Hello!';
  3. Finally use it:
    4. 
alert(message);

We can do first two steps in a 1 shot


var message =  'Hello!';

We can also change it as many times as we want:


let message;
message = 'Hello!';
message = 'World!'; // value changed
console(message);

We can also declare two variables and copy data from one into the other.


let hello = 'Hello World!';
let message;

// copy 'Hello world' from hello into message
message = hello;

// now two variables hold the same data
console.log(hello); //Hello world!
console.log(message); //Hello world!

Variable types

A variable in JavaScript can contain any data.

A variable can at one moment be a string and at another be a number:


let message = "hello";
console.log(message);
console.log(typeof(message));
message = 123456;
console.log(message);
console.log(typeof(message));

Numbers can include floating point numbers 


let n = 123;
n = 12.345;

Arrays

What is an Array?

An array is a special variable, which can hold more than one value at a time.

If you have a list of items (a list of car names, for example), storing the cars in single variables could look like this:


var car1 = "Saab";
var car2 = "Volvo";
var car3 = "BMW";

But you can imagine that in case of all brands might be inefficient. That's the best use case to use arrays:


let cars = ['Saab' , 'Volvo', 'BMW'];

How to access arrays elements?


console.log(cars[0]);
