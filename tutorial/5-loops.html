Loops

We often need to repeat actions.

For example, outputting goods from a list one after another or just running the same code for each number from 1 to 10.

Loops are a way to repeat the same code multiple times.

The “for” loop

Let's have a look at following example:


    for (let i = 0; i < 3; i++) { // shows 0, then 1, then 2
        alert(i);
    }

Loop schemat: 


    for (begin; condition; step) {
        // ... loop body ...
    }


    */
    begin       i = 0       Executes once upon entering the loop.
    condition   i < 3       Checked before every loop iteration. If false, the loop stops.
    body        alert(i)    Runs again and again while the condition is truthy.
    step        i++         Executes after the body on each iteration.

    i++ is a shorter veriosn of i = i + 1;  
    */

Example:


for (let i = 0; i < 20; i++) { // shows 0, then 1, then 2
    if(i%2 == 0) {
        console.log(i + " is even");
    }
}

Break

We can break loop at any moment , i.e. let's say we are searching for first number which can be divided by 9 and then stop


for (let i = 1; i < 120; i++) { // shows 0, then 1, then 2
    if(i%9 == 0) {
        console.log(i + " is even. My work is done here.");
        break;
    } else {
        console.log("Keep on searching.... checking: " + i)
    }
}

While

Another kind of loop is while() loop, its also quite simple,



    let i = 0;
        while (i < 3) { // shows 0, then 1, then 2
        console.log( i );
        i++;
    }

so as long a condition in while() is true, the loop will continue. It end when it becomes 0 (false)
