Why to use functions

Let's create our first function

alert( 'Hello everyone!' );
}

Lets copy this to our working file.

... and nothing happens. It's like with variable. You have to use it

and we can call it multiple times


showMessage();

Function naming

Functions are actions. So their name is usually a verb.
It should be brief, as accurate as possible and describe what the function does, so that someone reading the code gets an indication of what the function does.

For instance, functions that start with "show" usually show something.

Function starting with… 


    "get…" – return a value,
    "calc…" – calculate something,
    "create…" – create something,
    "check…" – check something and return a boolean, etc.
    
    showMessage(..)     // shows a message
    getAge(..)          // returns the age (gets it somehow)
    calcSum(..)         // calculates a sum and returns the result
    createForm(..)      // creates a form (and usually returns it)
    checkPermission(..) // checks a permission, returns */
